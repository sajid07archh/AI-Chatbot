# AI-Chatbot

## Description
An AI chatbot built using Google's Generative AI model that can answer user questions.

## Features
- Conversational AI chatbot
- Generates intelligent responses
- Built using Google Generative AI API
- Runs on Google Colab
- Simple user interface

## Tech Stack
- Python
- Google Generative AI
- Gradio
- Google Colab

## How to Run
1. Clone the repository
2. Add your API key
3. Run the notebook
